Building ViroCore --> ViroReact --> Use in starter kit

I am assuming that you have your directories set up with viro, virocore and the starter-kit as follows:

1. Build ViroCoreâ€‹

(iOS) Building the renderer to be used in react-viro platformâ€‹

1.1. Make sure you followed through steps 1 - 3 mentioned in the android sectionâ€‹

1.2. Execute the following commands to install pods from Podfileâ€‹

# in virocore/
cd ios
pod install

1.3. Open ViroRenderer.xcworkspace in Xcode. Build scheme ViroKit with Build Configuration set to Release and target set to Generic iOS Device.â€‹

Note:

If you want the ability to run on Simulator, change target to any of the iOS Simulator targets instead of Generic iOS Device. If the Xcode build succeeded, you should see a bunch of new files copied over in viro/ios/dist/ folder in the viro repo you cloned earlier.

2. Build ViroReactâ€‹

2.1. Execute the following commands to install pods from Podfileâ€‹

# in viro/
cd ios
pod install

2.2. Open ViroReact.xcworkspace in Xcode. Build scheme ViroReact with Build Configuration set to Release and target set to Generic iOS Device.â€‹

This should generate more files in the ios/dist/ folder.

2.3. Install podsâ€‹

I don't know if this is required, but I'm doing it anyways

# in viro/
cd ios
pod install

2.4. Bundle the ViroReact project into a tarballâ€‹

yarn bundle

3. Use in starter kitâ€‹

3.1. Clear the yarn cache of the tarballâ€‹

# I know, this cleans the entire cache. 
# yarn cache clean @viro-community/react-viro doesn't properly clear the tarball from the cache.
yarn cache clean

3.2. install dependenciesâ€‹

# in starter-kit/
yarn install

yarn add ../viro/viro-community-react-viro-<version>.tgz

cd ios
pod install

3.3. Run the starter kitâ€‹
